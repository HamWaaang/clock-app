# 移动端时钟应用

一个简单的移动端时钟应用，使用React 19 + TypeScript + Vite + Ant Design构建。

## 功能特性

- 🕐 实时显示当前时间
- 📅 显示当前日期
- ⏸️ 暂停/开始时钟
- 🔄 手动刷新时间
- 📱 移动端优化设计

## 技术栈

- React 19
- TypeScript
- Vite
- Ant Design
- CSS3

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
clock-app/
├── src/
```

## 移动端特性

- 响应式设计，适配各种屏幕尺寸
- 触摸友好的按钮设计
- 防止页面缩放
- 优化的字体和间距
